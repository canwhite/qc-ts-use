# qc-ts-use
ts的基础使用和一些帮助类型

## init

tsc --init  
* 运行完会出现tsconfig.json，可以对ts转译为js过程进行配置干预  
* ts只有esm的概念，对于前端，所有less和css文件要在typing里declare import，
    放入tsconfig的inlude才起作用  

## use

```
npm i ts-node -g
ts-node xxx.ts
```
